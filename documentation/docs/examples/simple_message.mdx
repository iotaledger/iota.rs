## Messages

A [message](../explanations/messages_payloads_and_transactions.md#messages) is an encapsulating data structure that is being actually broadcast across the network. It is an atomic unit that is either accepted or rejected as a whole.

The simplest message you can broadcast is a message without any particular payload, as shown in the following examples.
