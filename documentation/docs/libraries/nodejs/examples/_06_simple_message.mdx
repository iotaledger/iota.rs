You can use the function [`Client.postMessage(message)`](./../api_reference.md#postmessagemessage-promise) to send a message instance over a network. Alternatively, you can
use the [`MessageSender`](./../api_reference.md#messagesender) helper class with chaining calls that prepares a message instance and broadcasts it over the
network.
